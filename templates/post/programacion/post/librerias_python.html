Conozcamos algunas librerías para Python!

Creado por: Oliver Sandoval, 16/03/2018

Python es uno de los lenguajes de programación que ha adquirido aún mas popularidad de la que ya ha tenido a lo largo de todos los años que lleva con nosotros. Esto se debe ha su flexibilidad de uso, y mas ahora con el auge la Inteligencia Artificial, donde Python es el lenguaje que mas se usa para su desarrollo.

Librerías

Mencionaré algunas de las mas usadas para programar en este lenguaje. Estas son:

  • Request: La más famosa biblioteca ‘http’ escrito por Kenneth Reitz. Es una necesidad para todos los desarrolladores de Python.
  • Pillow: Una alternativa muy amistosa de PIL (Python Imaging Library). Es más fácil de usar que PIL y es una necesidad para cualquier persona que trabaja con imágenes.
  • NumPy: Proporciona algunas funcionalidades matemáticas muy útiles para el trabajo de Python Científico y el uso de AI con el mismo.
  • SciPy: Cuando hablamos de NumPy entonces tenemos que hablar de SciPy. Es una biblioteca de algoritmos y herramientas matemáticas para python y ha hecho que muchos científicos cambien de ruby a python.
  • Matplotlib: Una biblioteca para realizar gráficos. Es muy útil para cualquier científico de datos o cualquier analizador de datos.
  • Pygame: Biblioteca muy famosa, muy útil y para desarrollar videojuegos 2D.
  • PyQt: Es un binding de la biblioteca gráfica Qt para el lenguaje de programación Python, nos permite crear interfaces gráficas multiplataforma.
  • Kivy: Con esta grandiosa librería podrás realizar apps para Android, si haz jugado el juego 2048 verás que funciona y muy bien.
  • BioPython: Librería para el reconocimiento biométrico.
  • NLTK: Para el desarrollo del lenguaje natural humano, es muy amplia, muy util y muy potente.
  • Pyglet: Un motor de animación y creación de juego en 3D.
  • Nose: Un marco de pruebas para python. Es utilizado por millones de desarrolladores de Python. Es una necesidad si usted hace desarrollo basado en pruebas.
  • Sympy: Puede hacer evaluaciones algebraicas,diferenciación, numeros imaginarios entre otras cosas.
  • Scikit-learn: Scikit-learn es una biblioteca de aprendizaje automático de software libre para el lenguaje de programación Python. Incluye varios algoritmos de clasificación , regresión y agrupación que incluyen máquinas de vector de soporte, bosques aleatorios, aumento de gradiente, k- means y DBSCAN, y está diseñado para interactuar con las bibliotecas numéricas y científicas Python NumPy y SciPy.
  • Pandas: Pandas es una biblioteca de código abierto que proporciona estructuras de datos de alto rendimiento y fácil de usar y herramientas de análisis de datos para el lenguaje de programación Python. Su objetivo es ser el bloque fundamental de alto nivel para realizar análisis prácticos de datos en el mundo real en Python.
